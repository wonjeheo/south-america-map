# South America Route Map

남미 여행 중 가족에게 이동 경로와 체류 정보를 공유하기 위해 만든 웹 프로젝트입니다.

[Live Demo](https://wonjeheo.github.io/south-america-map/)

관리자는 도시, 일정, 이동 경로, 숙소와 비용 정보를 관리할 수 있으며, 게스트에게는 공개용으로 정리된 여행 경로만 제공합니다.

## Features

- Leaflet 기반 여행 경로 지도
- 도시 및 이동 경로 관리
- Google 계정 기반 관리자 인증
- 관리자 데이터와 게스트 공개 데이터 분리
- Firestore Security Rules 기반 접근 제어
- GitHub Pages 배포

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Map: Leaflet, OpenStreetMap
- Backend: Firebase Authentication, Cloud Firestore
- Deployment: GitHub Pages

## Purpose

남미 여행 중 가족에게 현재 이동 경로를 공유하고, 여행 기록을 한눈에 확인하기 위해 제작했습니다.
